# Setting up a MyndAtlas

(You may also be interested in [[How I got here]].)

Although the note files are all simply text files written markdown (and thus not locked in to any program or editor), the way I work with them is dependent on [Obsidian](https://obsidian.md/) and number of the availble plug-ins.

Note: My main devices are iOS based, which occasionally limits which plug-ins I can use, which then influences my overall way of working.

### Installing Obsidian
For iOS devices, Obsidian is available on the App Store. For MacOS I downloaded it from [Obsidian's website](https://obsidian.md/). On Ubuntu (on my [[Raspberry Pi 400]]), I installed Obsidian via [FlatPak](https://flathub.org/apps/details/md.obsidian.Obsidian).
