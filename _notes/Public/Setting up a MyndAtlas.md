# Setting up a MyndAtlas

Although the note files are all simply text files written markdown (and thus not locked in to any program or editor), the way I work with them is dependent on [Obsidian](https://obsidian.md/) and number of the availble plug-ins.

### Installing Obsidian
- For iOS devices, Obsidian is available on the App Store.
- For MacOS I downloaded it from [Obsidian's website](https://obsidian.md/).
- On my [[Raspberry Pi 400]], I'm running Obsidian through the ARM64 AppImage.
