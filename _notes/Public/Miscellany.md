# Miscellany

This is where other notes that don't fit with the larger knowledge work aspect of the note-system, or with the planner, branch off. Things like a recipe collection or the measurements of a window needed for ordering curtains or contact information for an insurance agent or something like that would go here.

Those familiar with "zettelkasten" or "Evergreen Notes" concepts might recognise this, along with the [[Planner]], as "outside" those systems. I still find it handy to have them accessible within Obsidian, so this is where I put them.
