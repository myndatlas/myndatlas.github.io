# ScratchBox
%%
- [ ] add something about home plugin
%%
This is currently my "home" page that opens when starting up Obsidian and is readily accessed via the home icon in the ribbon, using the homepage plugin.

The ScratchBox idea has evolved as something of an "inbox" for keeping track of tasks and items in progress in some way. For the most part, notes in the Scratchbox are here temporarily and the idea is to move notes out of here as soon as possible (and not sooner).

Most of the information on this note is generated by dataview queries so it updates automatically when changes are made. These come in several groups:

1. The [[ScratchPad]], [[Dataview Note]] and [[Query Note]].
	- These are three "fixed notes" that stay in the ScratchBox permanently for ease of access. Though seldom completely "empty", the aim is to clear the data off these notes at a reasonable pace, once it has been moved to a better location or is no longer needed.
2. Tasks lists imported from today and tomorrow's task lists on the current [[Planner]] pages.
3. Task Notes
	- These are notes that list tasks generated by queries. I'm still developing best practices for these - they are a way of gathering relevant tasks together and keeping irrelevant tasks out of the way.
4. General Task List
	- This is a list that excludes planner tasks and all the tasks captured by the Task Notes, which are used as filters to keep this managable.

5. A list of uncompleted tasks that don't fall within a query note. 
6. A list of any query notes that don't have a fixed home yet and may not be permanent.
7. A "net" to catch notes that haven't been connected to the Atlas to prevent lost or orphan notes.
